## featured work

### [WhyCode](https://github.com/Girishmanne6/whycode) — multi-agent codebase intelligence
`Python` `LangGraph` `RAG` `ChromaDB`

Four specialized agents run hybrid retrieval over dependency graphs, semantic search, and LLM reasoning across 1,100+ embeddings. Cuts codebase query time from 27s to 8.5s.

---

### [DriftShield](https://github.com/Girishmanne6/DriftShield) — real-time ML drift monitoring
`Python` `PyTorch` `WebSockets` `MLOps`

KS test + PSI + rolling accuracy with live WebSocket alerts. Implements TENT adaptation from scratch — recovers +18–27 pp accuracy under domain shift without retraining.

---

### [RankLab](https://github.com/Girishmanne6/RankLab) — responsible news recommendation
`Python` `LightGCN` `RecSys`

LightGCN with contrastive regularization on the 50K-user EB-NeRD subset. A/B dashboard tracking Recall, NDCG, and diversity side by side.

---

### [shift-left-compliance](https://github.com/Girishmanne6/shift-left-compliance) — policy-as-code DevSecOps
`OPA` `DevSecOps` `CI/CD`

Embeds compliance checks directly into the developer workflow — catches violations before they reach production, not after.
